# Vue Components

Scalable applications usually use components to organize UI components. You can nest them like this:

```html
<app>
  <toolbar>
  </toolbar>

  <content>
    <blog-post title="My journey with Vue"></blog-post>
    <blog-post title="Blogging with Vue"></blog-post>
  </content>

  <sidebar side="left">
    <toc><!-- ... --></toc>
    <tags><!-- ... --></tags>
    <ads><!-- ... --></ads>
  </sidebar>
</app>
```
